The QuickPoll application is designed to propose polls and register votes on your frontends.

Download {% endblock %} {% block github_ribbon %} {% endblock %} {% block content %}

Create a poll, add the questions, the choices, define the display rules questions, then start register votes.

The QuickPoll application is designed to not be intrusive, a ribbon is simply added on the left of your website if a poll is available, click it then vote !

Tips! You can replace the ribbon by a button, or display a button and a ribbon. Every elements with the open-poll class will open the QuickPoll window.

QuickPoll

settings.py

Add the application geelweb.django.quickpoll to your INSTALLED_APPS

INSTALLED_APPS = (
    ...
    'geelweb.django.quickpoll'
)
urls.py

Include the QuickPoll urls

urlpatterns = patterns('',
    url(r'^quickpoll/', include('geelweb.django.quickpoll.urls')),
)
Html template

Include the script to add the ribbon

<link href="{% templatetag openvariable %} STATIC_URL {% templatetag closevariable %}quickpoll/css/main.css" rel="stylesheet" media="screen" />
<script data-main="{% templatetag openvariable %} STATIC_URL {% templatetag closevariable %}quickpoll/js/main.js" src="{% templatetag openvariable %} STATIC_URL {% templatetag closevariable %}quickpoll/js/lib/vendor/require.js"></script>
